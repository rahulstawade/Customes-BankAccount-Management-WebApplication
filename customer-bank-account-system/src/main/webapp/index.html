Hi,
This is Rahul and I have built this project which is a basic online customer account management application.
There are 2 users in my project :-

  1. Bank's Employee
  2. Bank's Customer

There are several modules like :-

  • Adding New Customer
  • Adding New Account for Existing Customer's
  • Managing Customer's Account
  • Change Customer's Status
  • Registering Payee's
  • Money Transfer
  • Updating Profiles
  • Changing Password

I have used Several Technologies To develop this Application.
Some Of which are :- Java,Struts2,Hibernate,Spring,Bootstrap,MySQL,JSP,JQuery.
